# Mohd Nayyar - Next.js Portfolio

A modern, responsive portfolio website built with Next.js 14, featuring the exact same design as the original portfolio.html.

## 🚀 Features

- **Modern Tech Stack**: Next.js 14, React 18, CSS3
- **Responsive Design**: Mobile-first approach with smooth animations
- **Interactive Navigation**: Smooth scrolling between sections
- **AI Chat Assistant**: Integrated with static responses for portfolio inquiries
- **Component-Based Architecture**: Modular, maintainable code structure
- **SEO Optimized**: Meta tags and semantic HTML
- **Performance Optimized**: Fast loading and smooth interactions

## 🛠️ Tech Stack

- **Framework**: Next.js 14 with App Router
- **Frontend**: React 18, JavaScript ES6+
- **Styling**: CSS3 with custom properties, Flexbox, Grid
- **Icons**: Font Awesome 6.4.0
- **Fonts**: Google Fonts (Inter)
- **Deployment**: Static export ready for Vercel, Netlify, etc.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/personal-nayyar/nextjs-portfolio.git
cd nextjs-portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📜 Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run export    # Export static site
npm run lint      # Run ESLint
```

## 🚀 Deployment

### Vercel (Recommended)
1. Push to GitHub
2. Connect repository to [Vercel](https://vercel.com)
3. Automatic deployment on every push

### Netlify
1. Build the project: `npm run build`
2. Export static: `npm run export`
3. Upload `out` folder to Netlify

### Static Export
```bash
npm run build
npm run export
```
The static files will be in the `out` directory.
